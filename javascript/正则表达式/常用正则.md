# 基础

```
  正整数或者保留2位小数      /^\d+(\.\d{2})?$/
```

```
  座机号码或手机号码验证     /^(((\d{3,4}-)?\d{8})|(1[3568]\d{9}))$/ 
```

```
  数字  1 ~ 999            /^[1-9]\d{0,2}$/
```

```
  身份证                   /^\d{15}(?:\d{2}[\d|X|x])$/
```

```
  去除keyword两边的所有空格  var keyword = keyword.replace(/(^\s+)|(\s+$)/g,"")   
```

```
  将keyword 中％转义        var keyword = keyword.replace(/%+/,'\\%');    
```
