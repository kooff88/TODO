## nav

html5新标签

定义导航链接的部分    

并不是所有的html文档都要使用到<nav>元素.<nav>元素只是作为标注一个导航链接的区域。   
在不同的设备上(手机或者pc)可以制定导航链接是否显示，已适应不同屏幕的需求.  

```
  <nav>
    <a href="/html/">HTML</a> |
    <a href="/css/">CSS</a> |
    <a href="/js/">JavaScript</a> |
    <a href="/jquery/">jQuery</a>
  </nav>
```

## figure

```html
</figure>
```
